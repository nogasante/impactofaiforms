# # Impact of AI on Students' Learning Habits

A lightweight, single-page web survey exploring how Artificial Intelligence tools
are shaping the way students study, solve problems, and manage their academic work.

The page introduces the research, then loads an embedded Google Form on demand so
responses can be collected directly from the site.

🔗 **Live site:** https://nogasante.github.io/impact-of-ai/

---

## Features

- **Tap-to-load survey** — the Google Form only loads when the visitor taps
  *Take the survey*, so the page opens instantly.
- **Fully responsive** — scales smoothly from desktop down to the smallest phones.
- **Minimal, tasteful animations** — gentle fade-ins and hover effects, with
  `prefers-reduced-motion` respected.
- **Anonymous & neutral** — no personal or course details, so responses stay impartial.
- **Single file, no dependencies** — everything (styles, logo, script) lives in
  `index.html`. Nothing to install or build.

---

## Tech

- HTML, CSS, and a small amount of vanilla JavaScript
- Google Forms (embedded via iframe)
- No frameworks, no build step

---

## How to run

Just open `index.html` in any browser — that's it.

To host it for free with GitHub Pages:

1. Go to the repo **Settings → Pages**.
2. Under **Branch**, choose `main` and folder `/ (root)`, then **Save**.
3. Wait about a minute. The site goes live at the link above.
